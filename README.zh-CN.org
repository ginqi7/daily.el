* 概述

daily.el 是一个基于 [[https://github.com/ginqi7/emacs-ctable][ctable]] 和 [[https://github.com/magit/emacsql][emacsql]] 的 Emacs 日记/日志管理插件。它提供了完整的日记管理功能：

- 创建、编辑和删除条目
- 基于标签的过滤和组织
- 支持分页浏览，可自定义每页大小
- SQLite 数据库后端，持久化存储
- 交互式仪表板，带过滤按钮
- Org-mode 集成，用于条目内容

* 安装

** 方法一：Git 克隆

克隆仓库到你的 Emacs 配置目录：

#+BEGIN_SRC sh
  git clone https://github.com/ginqi7/daily ~/.emacs.d/lisp/daily
#+END_SRC

在 Emacs 配置中添加路径到 =load-path=：

#+BEGIN_SRC emacs-lisp
  (add-to-list 'load-path "~/.emacs.d/lisp/daily")
  (require 'daily)
#+END_SRC

** 方法二：使用 use-package

#+BEGIN_SRC emacs-lisp
  (use-package daily
    :load-path "~/.emacs.d/lisp/daily"
    :commands daily)
#+END_SRC

* 使用方法

** 启动 daily

#+BEGIN_SRC emacs-lisp
  M-x daily
#+END_SRC

或在 Emacs 配置中添加按键绑定：

#+BEGIN_SRC emacs-lisp
  (global-set-key (kbd "C-c d") #'daily)
#+END_SRC

* 自定义配置

** 自定义变量

| 变量                | 默认值               | 说明                   |
|---------------------+----------------------+------------------------|
| =daily-time-format= | ="%Y-%m-%d %H:%M:%S"= | 日期时间显示格式       |
| =daily-page-size=   | =30=                  | 每页加载的条目数量     |
| =daily-db-path=     | =~/daily.db=          | SQLite 数据库文件路径  |
| =daily-db-limit=    | =100=                 | 数据库中最大记录数     |

** 配置示例

#+BEGIN_SRC emacs-lisp
  (require 'daily)

  ;; 自定义日期时间格式
  (setq daily-time-format "%Y/%m/%d %H:%M")

  ;; 设置每页 50 条
  (setq daily-page-size 50)

  ;; 自定义数据库路径
  (setq daily-db-path "~/.emacs.d/daily.db")
#+END_SRC

* 数据结构

** daily-one (条目)

#+BEGIN_EXAMPLE
  daily-one
  ├── uuid      (string)   唯一标识符 (UUIDv4)
  ├── date      (string)   创建时间
  ├── text      (string)   正文内容
  └── tags      (list)     标签列表
          └── daily-tag
              ├── uuid      (string)  标签唯一标识符
              ├── name      (string)  标签名称
              └── one-uuid  (string)  所属条目 UUID
#+END_EXAMPLE

** 数据库结构

*** 表：one

| 列名    | 类型    | 说明                     |
|---------+---------+--------------------------|
| uuid    | TEXT    | 主键                     |
| date    | TEXT    | 时间戳                   |
| text    | TEXT    | 内容                     |
| deleted | INTEGER | 软删除标志 (0/1)         |

*** 表：tag

| 列名     | 类型 | 说明                               |
|----------+------+------------------------------------|
| uuid     | TEXT | 主键                               |
| name     | TEXT | 标签名称                           |
| one-uuid | TEXT | 外键，引用 one(uuid)               |

* API 参考

** 核心函数

| 函数                                    | 说明                           |
|-----------------------------------------+--------------------------------|
| =daily=                                 | 启动 daily 界面                |
| =daily-add=                             | 新增条目                       |
| =daily-edit=                            | 编辑选中条目                   |
| =daily-delete=                          | 删除选中条目                   |
| =daily-show=                            | 在文本缓冲区打开条目           |
| =daily-preview=                         | 预览条目                       |
| =daily-refresh=                         | 刷新界面                       |
| =daily-accumulate=                      | 以 org 格式汇总条目            |
| =daily-set-filter=                      | 按选中列设置过滤               |

* 故障排除

** 常见问题

*** 问题：无法打开加载文件：ctable

解决方案：确保 ctable 已安装并添加到 load-path。

*** 问题：数据库文件被锁定

解决方案：关闭其他使用相同数据库文件的 Emacs 实例。
