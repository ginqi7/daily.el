* Overview

A powerful Emacs daily/journal management package built on [[https://github.com/ginqi7/emacs-ctable][ctable]] and [[https://github.com/magit/emacsql][emacsql]]. It provides a complete solution for managing your daily entries with features including:

- Create, edit, and delete entries
- Tag-based filtering and organization
- Paginated browsing with customizable page size
- SQLite database backend for persistent storage
- Interactive dashboard with filter buttons
- Org-mode integration for entry content

Run the dependencies script:

#+BEGIN_SRC sh
  ./dependencies.sh
#+END_SRC

* Installation

** Method 1: Git Clone

Clone the repository to your Emacs configuration directory:

#+BEGIN_SRC sh
  git clone https://github.com/ginqi7/daily ~/.emacs.d/lisp/daily
#+END_SRC

Add the path to your =load-path= in Emacs config:

#+BEGIN_SRC emacs-lisp
  (add-to-list 'load-path "~/.emacs.d/lisp/daily")
  (require 'daily)
#+END_SRC

** Method 2: Using use-package

#+BEGIN_SRC emacs-lisp
  (use-package daily
    :load-path "~/.emacs.d/lisp/daily"
    :commands daily)
#+END_SRC

* Usage

** Start daily

#+BEGIN_SRC emacs-lisp
  M-x daily
#+END_SRC

Or add a key binding in your Emacs config:

#+BEGIN_SRC emacs-lisp
  (global-set-key (kbd "C-c d") #'daily)
#+END_SRC

* Key Bindings

** Main Interface

| Key    | Function                   |
|--------+----------------------------|
| =a=    | Add new entry              |
| =e=    | Edit selected entry        |
| =d=    | Delete selected entry      |
| =g=    | Refresh/Accumulate         |
| =f=    | Set filter by column       |
| =SPC=  | Preview entry              |
| =RET=  | Open entry in new buffer   |
| =+=    | Load more (next page)      |
| =q=    | Quit                       |

** Text Edit Mode

| Key         | Function               |
|-------------+------------------------|
| =C-c C-c=   | Submit edited text     |

* Configuration

** Custom Variables

| Variable            | Default Value        | Description                  |
|---------------------+----------------------+------------------------------|
| =daily-time-format= | ="%Y-%m-%d %H:%M:%S"= | Date/time display format     |
| =daily-page-size=   | =30=                  | Number of entries per page   |
| =daily-db-path=     | =~/daily.db=          | Path to the SQLite database  |
| =daily-db-limit=    | =100=                 | Maximum records in database  |

** Configuration Example

#+BEGIN_SRC emacs-lisp
  (require 'daily)

  ;; Custom date/time format
  (setq daily-time-format "%Y/%m/%d %H:%M")

  ;; Set page size to 50 entries
  (setq daily-page-size 50)

  ;; Custom database path
  (setq daily-db-path "~/.emacs.d/daily.db")
#+END_SRC

* Data Structure

** daily-one (Entry)

#+BEGIN_EXAMPLE
  daily-one
  ├── uuid      (string)   Unique identifier (UUIDv4)
  ├── date      (string)   Creation timestamp
  ├── text      (string)   Text content
  └── tags      (list)     List of tags
          └── daily-tag
              ├── uuid      (string)  Tag unique ID
              ├── name      (string)  Tag name
              └── one-uuid  (string)  Parent entry UUID
#+END_EXAMPLE

** Database Schema

*** Table: one

| Column  | Type    | Description                    |
|---------+---------+--------------------------------|
| uuid    | TEXT    | Primary key                    |
| date    | TEXT    | Timestamp                      |
| text    | TEXT    | Content                        |
| deleted | INTEGER | Soft delete flag (0/1)         |

*** Table: tag

| Column   | Type | Description                              |
|----------+------+------------------------------------------|
| uuid     | TEXT | Primary key                              |
| name     | TEXT | Tag name                                 |
| one-uuid | TEXT | Foreign key referencing one(uuid)        |

* API Reference

** Core Functions

| Function                              | Description                              |
|---------------------------------------+------------------------------------------|
| =daily=                               | Start the daily interface                |
| =daily-add=                           | Add a new entry                          |
| =daily-edit=                          | Edit selected entry                      |
| =daily-delete=                        | Delete selected entry                    |
| =daily-show=                          | Open entry in text buffer                |
| =daily-preview=                       | Preview entry                            |
| =daily-refresh=                       | Refresh the interface                    |
| =daily-accumulate=                    | Accumulate entries in org format         |
| =daily-set-filter=                    | Set filter by selected column            |
| =daily-set-filter-date=               | Set date filter                          |
| =daily-set-filter-text=               | Set text filter                          |
| =daily-set-filter-tags=               | Set tags filter                          |

** Database Functions

| Function                                     | Description                                |
|----------------------------------------------+--------------------------------------------|
| =daily-db-init=                              | Initialize database tables                 |
| =daily-db-one-insert-or-update=              | Insert or update a one entry               |
| =daily-db-tag-insert-or-update=              | Insert or update a tag                     |
| =daily-db-one-insert-or-update-with-tags=    | Insert or update entry with tags           |
| =daily-db-get-one=                           | Get entry by UUID with tags                |
| =daily-db-list-one-uuid=                     | List entry UUIDs with filters              |
| =daily-db-delete-one=                        | Soft delete an entry                       |
| =daily-db-delete-one-with-tags=              | Soft delete entry and remove tags          |
| =daily-db-delete-tags=                       | Delete tags for an entry                   |
| =daily-db-one-count=                         | Count non-deleted entries                  |
| =daily-db-no-repeat-tag-names=               | Get distinct tag names                     |

* Troubleshooting

** Common Issues

*** Issue: "Cannot open load file: ctable"

Solution: Ensure ctable is installed and added to load-path.

*** Issue: Database file is locked

Solution: Close any other Emacs instances using the same database file.
